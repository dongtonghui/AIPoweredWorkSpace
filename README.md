## 本地查看

直接双击 `index.html`，或使用任意静态服务器：

```bash
python3 -m http.server 8000
```

然后访问 http://localhost:8000

## 结构

- `index.html` — 演示入口（概览墙 + 全屏演讲模式）
- `slides/` — 19 页独立幻灯片
- `design-demos/` — 设计方向探索稿

## 操作说明

- 点击任意幻灯片卡片进入演示模式
- `← / →` 翻页
- `Esc` 退出到概览墙
