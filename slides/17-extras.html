MORE TOOLS

其它 GitHub 协作工具：不止于代码和文档

🐛
Issues
把需求、bug、调研任务都变成可追踪的 ticket。每个 Issue 可以关联 PR、文档、原型链接。
适合场景
任务分发、缺陷跟踪、技术调研、上线 checklist。
💬
Discussions
开放式论坛，适合做方案头脑风暴、Q&A、决策记录。不像 Issue 那样需要 close。
适合场景
“首页推荐用协同过滤还是规则引擎？”这类需要充分讨论的决策。
📊
Projects
看板/表格视图管理迭代进度。把 Issues、PR、文档任务串成一条交付流水线。
适合场景
迭代规划、跨职能协作、对外汇报进度。
这些工具 + Markdown 文档 + HTML 原型，构成一个轻量级但完整的产研协作 OS。
